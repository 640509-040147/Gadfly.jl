Guide.annotation

Guide.annotation

Overlay a plot with an arbitrary Compose graphic. The context will inherit the plot's coordinate system, unless overridden with a custom unit box.

Arguments

  • ctx: A Compose Context.

Examples

plot(sin, 0, 2pi,
     Guide.annotation(
       compose(context(), Shape.circle([pi/2, 3*pi/2], [1.0, -1.0], [2mm]), fill(nothing),
       stroke("orange"))))
