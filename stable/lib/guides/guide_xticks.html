Guide.xticks

Formats the tick marks and labels for the x-axis

Arguments

  • ticks: Array of tick locations on the x-axis, :auto to automatically select ticks, or nothing to supress x-axis ticks.

  • label: Determines if the ticks are labeled, either true (default) or false

  • orientation: Label orientation (:horizontal, :vertical, :auto). Defaults to :auto

Examples

ticks = [0.1, 0.3, 0.5]
plot(x=rand(10), y=rand(10), Geom.line, Guide.xticks(ticks=ticks))
plot(x=rand(10), y=rand(10), Geom.line, Guide.xticks(ticks=ticks, label=false))
plot(x=rand(10), y=rand(10), Geom.line, Guide.xticks(ticks=ticks, orientation=:vertical))
